# TIPE-Lidar

Suite à notre passage en prépa TSI, nous avons réalisé un TIPE sur le thème "boucle et cycle". Notre était "Comment reperer un robot dans une pièce à partir d'un capeur lidar ?".

Nous nous sommes servi d'un capteur lidar RP A1 que nous avons associé avec python pour traiter les données que nous avons aquis.

Ici vous retrouverez les codes qui nous ont permis de mettre en oeuvre notre TIPE.
